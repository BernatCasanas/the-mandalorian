All about Diamond Engine

Diamong Engine is a lightweight 3D (and future 2D) game development engine. This engine was developed as part of a collage assigment. The engine supports almost every editor feature that you could expect from a game development engine, but the most relevant feature is the added support to interpret and work with C# code and a component based GameObject system.

The C# support was possible thanks to the mono runtime.
